# ProyectoSoyJere

Torneo de combates multijugador para jugar con amigos: 8 luchadores, builds personalizados y narración automática.

## Stack (planificado)

- React + Vite + TypeScript
- Express + Socket.io (polling)
- Deploy en Hostinger Cloud Startup
